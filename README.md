# AHK Scripts for Devs

AHK (Auto Hot Key) scripts for Devs

## Repo

### 1. Clone

On `~` (home) path clone this repo

```bash
git init 
git remote add origin https://github.com/deinsoftware/ahk-scripts-devs.git
git pull origin main
```
